case class GenotypeCall(genotype: IndexedSeq[Option[Int]], ad: IndexedSeq[Int], aq: IndexedSeq[Int], pl: IndexedSeq[IndexedSeq[Int]]) extends Product with Serializable

A class to store genotype information for a single sample/cell

genotype

Alleles of the sample, None means not called

ad

Depth of each allele

aq

Quality of each allele in phred

pl

Pgred value for each allele and each possible expected alleles, for ploidy 2 that should be 3 values, 0,1,2
